기구물 비전 검사.html — 개발 맥락 분석
제품 정의
웹 기반 실시간 AI 비전 검사 시스템 — 제조 현장의 기구물(기구 부품/조립품) 외관 검사를 브라우저 단독 실행으로 수행하는 툴. 별도 서버 없이 HTML 파일 하나로 완결됩니다.

핵심 기술 스택
레이어기술비전 처리OpenCV.js 4.8.0 (WASM)딥러닝 AITensorFlow.js + MobileNet v2차트Chart.js엑셀 내보내기XLSX.js스토리지localStorage + File System Access APIUI순수 HTML/CSS (Dark 제조현장 테마)

기능 진화 맥락 (리버스 엔지니어링)
초기 버전 (기초 기능)

웹캠 연결 → ROI 드래그 지정 → 양품 이미지 1장 등록 → 템플릿 매칭(TM_CCOEFF_NORMED) → OK/NG 판정
단순 matchTemplate 전체 탐색 방식

확장 1: 다중 양품 + 바코드 자동 매칭

masterMats[] 배열로 다중 마스터 관리
바코드 리더기 입력 → 5~14번째 자리 모델 코드 추출 → masterFileRegistry에서 자동 매칭 → 해당 모델 양품 이미지로 자동 전환
양품 폴더 선택(FileSystem Access API) → 이미지 일괄 로드

확장 2: ROI 고정 비교 방식 전환

matchTemplate 전체 탐색 방식 → ROI 좌표 직접 고정 비교로 변경 (코드 주석에 명시)
savedRoi 개념 추가: 양품 등록 시 ROI 좌표를 함께 저장
자석 윤곽선 ROI (auto_contour): Canny Edge → Dilate/Erode → 최대 윤곽선 추출 → 마스크 생성

확장 3: 딥러닝 AI 교차 판정

MobileNet 임베딩 벡터 추출 → Cosine Similarity로 CV 결과와 이중 판정
dlEmbedding 저장 및 비교 옵션화 (체크박스로 활성화/비활성화)

확장 4: 검사 워크플로우 고도화

5프레임 안정화 → 24프레임(0.8초) 연속 OK 확인 → 판정 확정 로직
OK 판정 시 화면 동결(isFrozen) → 자동 이미지 저장 → 바코드 팝업 → 다음 제품 검사 연속 진행
이력 로그 → XLSX 다운로드 (검사이력/요약 시트)


핵심 설계 특이점
판정 로직 (3단계 방어)
1. stableFrames < 5  → STABILIZING... (위치 안정화 대기)
2. continuousOkFrames < 24 → CHECKING... (0.8초 지속 확인)
3. 확정 → OK/NG 기록 + 이미지 자동 저장
바코드 기반 생산 연동 흐름
바코드 스캔 → 모델 코드 추출 → 양품 이미지 자동 로드 
→ 검사 자동 시작 → OK 시 이미지 저장 → 바코드 팝업 
→ 다음 제품 바코드 스캔 → 반복
중복 카운트 방지 설계

inspectionResultLogged, ignoreOkUntilRemoved, currentResultIsOK 3개 플래그로 동일 제품 이중 기록 차단
NG→OK 업그레이드 로직 (제품 치우기 전 재판정 허용)
